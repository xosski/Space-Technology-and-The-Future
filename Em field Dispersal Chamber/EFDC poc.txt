EM Field Dispersal Chamber PoC (Defensive)
Goal

Show that an “alternating cell liner” inside a conductive chamber reduces how much an external transient/EM noise couples into an inner protected volume.

You’ll compare 3 builds and measure induced spikes inside:

Plain chamber (baseline Faraday-ish box)

Chamber + alternating dispersal liner (“PTEC cells”)

Chamber + liner + absorption (ferrite sheet/tiles)

Concept (your “alternating PTECs” translated)

Your “two alternating PTECs on the side” becomes:

Alternating conductive segments on opposing walls

Segments are not one continuous sheet (segmentation breaks big eddy-current loops)

Optional: segments connect to the shell via bleed paths (slow discharge, reduces charge build-up)

Optional: add lossy material (ferrite) in corners to damp resonant modes

This is “field dispersal” because you’re preventing stable standing waves and forcing induced currents into smaller, less effective loops.

Materials (easy to get)
Chamber + inner volume

Outer chamber: metal ammo can / metal toolbox / cookie tin / steel file box

Inner “null box”: small metal project box OR copper tape box OR mesh pouch

Plastic standoffs / foam blocks to keep inner box from touching outer chamber

Alternating dispersal liner (the “cells”)

Copper tape (conductive adhesive preferred) or aluminum tape

Thin plastic sheet/backing (optional, helps alignment)

(Optional) 8–16 resistors, high value (e.g., 100k–1M) for bleed paths

Absorption add-on (optional)

Ferrite sheet (EMI shielding sheet) or clip-on ferrites placed near penetrations

Measurement payload

A tiny microcontroller logger (or any simple device that can log resets)

A “probe loop”: 5–20 turns of wire loop inside the inner box

Something to read probe:

oscilloscope (best), or

microcontroller ADC logger, or

cheap USB sound card input (crude but works for relative comparison)

Stimulus source (safe)

ESD tester (preferred), or

benign EMC noise/transient source

If neither: controlled repeatable “static discharge” to the OUTER chamber only (least repeatable)

How to build the “alternating dispersal liner”

Pick one wall, then the opposite wall.

Cut 8–16 vertical strips of copper tape (same length).

Apply strips with small gaps between them (2–5 mm).

On the opposite wall, apply strips offset (so seams don’t align).

Keep strips isolated from each other (don’t bridge them at the ends).

Optional bleed paths (recommended for “PTEC vibe”):

Connect each strip to the chamber wall through a high-value resistor (slow discharge path).

This keeps strips from floating electrically while still preventing one giant loop.

Procedure (3 conditions, same stimulus)

Choose N = 20 hits per condition (or 50 if you want better stats).

Condition A — Plain chamber

Inner null box + payload inside outer chamber

No liner

Apply N hits to the same marked spot on the outer chamber

Record:

payload resets? (Y/N)

probe peak amplitude

Condition B — Add dispersal liner

Add alternating segmented strips

Same N hits, same spot, same spacing

Record metrics

Condition C — Add absorption

Keep liner, add ferrite near corners/seams and near any cable penetration

Same N hits

Record metrics

What counts as “working”

You’re looking for:

Lower peak on the probe loop inside the null box (B < A, C < B ideally)

Fewer resets/glitches (especially between B and C)

Even if B doesn’t lower peaks dramatically, it may reduce rare worst-case spikes (that’s still a win).

Tuning knobs (if you don’t see improvement)

Increase strip count (8 → 16)

Increase offset between opposite walls

Reduce alignment of seams between inner/outer boxes

Improve penetrations: cables are usually the leak path (shorten, add ferrites)

Add ferrite near corners (corners love resonances)
