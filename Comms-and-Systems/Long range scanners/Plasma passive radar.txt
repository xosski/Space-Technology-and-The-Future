PoC Draft
VeilPiercer Passive-First Radar Mode
Drift-Pulse Interference Ranging and Thermal Confirmation

Version: v0.3 PoC
Date: 2026-03-05
Program: Project VeilPiercer / GhostScope Integration

1) Abstract

This Proof-of-Concept proposes a passive-first sensing architecture that combines baseline environmental listening with optional low-energy probe pulses to detect, localize, and classify anomalies in a monitored volume. Unlike conventional radar, the system is designed to first establish a coherence map of the environment, then identify disturbances through shifts in phase, attenuation, timing, and thermal drift. Only after passive anomaly detection does the system enter a limited active confirmation mode, using short probe pulses to estimate range, shape, and persistence. This aligns with the VeilPiercer concept of layered interference capture, the passive/active thermal detection model, and the GhostScope-1 structural confirmation approach.

2) Objective

Demonstrate a sensing stack that can:

passively detect anomalous disturbances in an environment,

distinguish persistent objects from transient noise,

optionally apply low-pulse interrogation for confirmation,

correlate signal anomalies with thermal or structural signatures,

produce a repeatable confidence score for operator review.

This PoC is intended for defensive sensing, monitoring, and environmental awareness only.

3) Concept Summary
Tier 1 — Passive Drift Listening

The system continuously listens to background environmental fields and builds a baseline map of:

interference patterns,

harmonic drift,

phase coherence,

thermal background stability.

This matches the “passive harmonic drift” and baseline anomaly detection ideas in the VeilPiercer material.

Tier 2 — Low-Pulse Interrogation

When passive monitoring detects a candidate anomaly, the system emits a minimal confirmation pulse into the monitored volume and measures:

return delay,

phase distortion,

attenuation gap,

shadowing,

wake persistence.

This is the “active confirmation” layer already implied in the VeilPiercer PoC.

Tier 3 — Secondary Confirmation

If the anomaly remains unresolved, a second sensing channel is used:

thermal differential confirmation,

structural/diffraction-style confirmation,

or high-speed waveform comparison.

This aligns with GhostScope-1 and the passive/active thermal detection concept.

4) Technical Hypothesis

A monitored environment has a measurable baseline pattern. When an object, disturbance, or structural anomaly enters the detection space, it changes that pattern in ways that can be measured before and after low-energy pulse interrogation.

The system should be able to detect anomalies by combining:

passive coherence breaks,

active pulse return differences,

and secondary thermal/structural confirmation.

The expected result is a hybrid passive/active ranging system with better false-positive resistance than passive-only sensing.

5) PoC Architecture
A) Sensor Node Array

A multi-node listening geometry surrounding or facing the test zone.

Functions:

capture ambient signal drift,

compare node-to-node phase differences,

triangulate anomaly directionality.

B) Pulse Source

A controlled, low-energy pulse generator used only after passive detection.
Purpose:

confirm range,

confirm persistence,

improve shape estimate.

C) Thermal Confirmation Channel

Independent thermal sensing to detect:

wake effects,

thermal voids,

surface heat anomalies.

D) Structural Confirmation Channel (Optional)

A higher-resolution confirmation layer inspired by GhostScope-style diffraction/structural probing for difficult targets.

6) PoC Test Phases
Phase 1 — Passive Baseline Mapping

Goal: establish a clean environmental signature with no target present.

Procedure:

run sensors continuously over fixed intervals,

record harmonic drift, phase stability, and thermal background,

generate a baseline interference/coherence map.

Measurements:

phase variance,

background noise spectrum,

thermal stability,

cross-node timing offsets.

Success criteria:

stable baseline map,

repeatable node correlation over time.

Phase 2 — Passive Anomaly Detection

Goal: prove that introduced objects create detectable passive deviations.

Procedure:

introduce known test objects into the monitored volume,

compare passive readings against baseline,

classify transient vs persistent changes.

Measurements:

coherence break magnitude,

anomaly duration,

multi-node agreement,

thermal drift deviation.

Success criteria:

known objects produce measurable anomaly signatures,

false positive rate remains bounded.

Phase 3 — Low-Pulse Confirmation

Goal: confirm that brief interrogation pulses improve detection confidence.

Procedure:

after passive anomaly detection, emit a short probe pulse,

measure response timing, attenuation, and shadowing,

compare against passive-only classification.

Measurements:

pulse delay,

phase distortion,

return attenuation,

confidence score improvement.

Success criteria:

low-pulse mode improves target localization,

active confirmation reduces ambiguous classifications.

Phase 4 — Thermal / Secondary Confirmation

Goal: validate that thermal or structural signatures help confirm difficult targets.

Procedure:

repeat Phase 2–3 with thermal channel enabled,

compare classification performance with and without thermal confirmation.

Measurements:

thermal delta map,

wake persistence,

cross-correlation with pulse-confirmed anomaly position.

Success criteria:

improved distinction between real targets and noise,

better confidence on low-signature objects.

7) Null Tests

These are required for credibility.

No target / passive only → baseline must remain stable

Target present / passive only → anomaly must appear before active confirmation

Pulse with no target → system must not invent false structure

Thermal-only disturbance → should not be misclassified as geometric target without corroboration

Transient noise event → must decay below persistence threshold

8) Deliverables

baseline coherence maps,

anomaly event logs,

pulse-confirmation range curves,

thermal correlation overlays,

false-positive / false-negative analysis,

recommended threshold set for passive-to-active handoff.

9) What This PoC Proves

Proves:

passive-first anomaly detection is viable,

low-pulse interrogation can improve range/confidence,

thermal/secondary channels reduce ambiguity,

a hybrid passive/active detection suite can be built as a layered system.

Does not prove:

full conventional radar replacement in all environments,

perfect classification,

long-distance performance without scale-up testing.
