7. Control Logic
def rez_escape_burn_step(ship, rez_top, rez_bottom, lattice, command):
    """
    Speculative control step for REZ Lazarus Emergency Escape Burn.
    """

    # 1. Read ship state
    velocity = ship.velocity_vector
    altitude = ship.altitude
    gravity = ship.local_gravity
    hull_stress = ship.hull_stress
    crew_g_load = ship.crew_g_load

    # 2. Read energy state
    top_charge = rez_top.available_energy
    bottom_charge = rez_bottom.available_energy
    lattice_margin = lattice.available_absorption
    thermal_margin = min(rez_top.thermal_margin, rez_bottom.thermal_margin)

    # 3. Validate emergency burn conditions
    if not command.emergency_authorized:
        return hold_fire(reason="escape_burn_requires_emergency_authorization")

    if abs(top_charge - bottom_charge) > command.max_charge_imbalance:
        return hold_fire(reason="top_bottom_charge_imbalance")

    if lattice_margin < command.minimum_lattice_absorption:
        return hold_fire(reason="insufficient_lattice_backlash_capacity")

    if thermal_margin < command.minimum_thermal_margin:
        return hold_fire(reason="thermal_margin_low")

    if hull_stress > ship.max_hull_stress * 0.80:
        return hold_fire(reason="hull_stress_too_high")

    if crew_g_load > ship.max_crew_g_load:
        return hold_fire(reason="crew_acceleration_limit")

    # 4. Calculate burn vector
    burn_vector = calculate_escape_vector(
        current_velocity=velocity,
        gravity=gravity,
        target_trajectory=command.target_trajectory,
        obstacle_map=ship.local_obstacles,
    )

    # 5. Set both arrays to same-direction discharge
    rez_top.align_discharge_vector(burn_vector)
    rez_bottom.align_discharge_vector(burn_vector)

    # 6. Calculate permitted pulse energy
    permitted_energy = min(
        command.requested_pulse_energy,
        top_charge,
        bottom_charge,
        command.max_energy_per_burn,
    )

    # 7. Fire synchronized pulse
    pulse_plan = synchronized_escape_pulse(
        vector=burn_vector,
        energy=permitted_energy,
        duration_ms=command.duration_ms,
        phase_offset_ms=0,
    )

    rez_top.discharge(pulse_plan.top)
    rez_bottom.discharge(pulse_plan.bottom)

    # 8. Estimate impulse and update trajectory model
    impulse = estimate_escape_impulse(pulse_plan, ship.mass)
    ship.velocity_vector += impulse / ship.mass

    # 9. Absorb backlash
    lattice.absorb_transient_load(
        electrical_spike=pulse_plan.estimated_backlash,
        thermal_spike=pulse_plan.estimated_heat,
        field_echo=pulse_plan.estimated_field_echo,
    )

    # 10. Enter cooldown/recharge lockout
    rez_top.lockout_until_safe()
    rez_bottom.lockout_until_safe()

    return {
        "mode": "REZ_ESCAPE_BURN",
        "burn_vector": burn_vector,
        "impulse": impulse,
        "new_velocity": ship.velocity_vector,
        "lattice_margin": lattice.available_absorption,
        "status": "burn_executed",
    }
8. Math Sketch
Required impulse
J = Δp = m Δv

Where:

J = impulse
m = ship mass
Δv = desired velocity change
Average thrust
F = J / Δt

Where:

F = average thrust
Δt = burn duration
Kinetic energy change
ΔE ≈ 1/2 m(v₂² - v₁²)

For emergency boosts, energy requirements become brutal fast.

This is why REZ Escape Burn is best as a short assist, not a full launch engine.

9. Escape Velocity Reality Check

Escape velocity from a body is:

v_escape = sqrt(2GM / r)

Where:

G = gravitational constant
M = mass of the planet/body
r = distance from center of mass

But in practical ship terms, the burn does not need to instantly reach escape velocity if:

- main engines are also operating
- the ship already has orbital velocity
- the burn only prevents atmospheric loss
- the burn raises periapsis
- the burn buys time for reactor restart

So REZ Escape Burn should be framed as:

emergency delta-v injection
gravity-well assist
trajectory rescue pulse

Not “single-handedly launches battleship from planet forever.”

Unless the ship is tiny or the capacitors are absurd. And when capacitors are absurd, funerals become architectural.
