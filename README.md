## Navigation Guide

Use these entry points to start browsing quickly:

1. `Energy-and-Power/Genesis Reactor/` for core reactor narratives and integration notes.
2. `Propulsion-and-Vehicles/Propulsion Systems/` for the largest propulsion cluster.
3. `Shielding-and-Defense/Reactive Em Shielding/` for EM-focused defense material.
4. `Comms-and-Systems/Long range scanners/` for detection and scanner concepts.
5. `Medical/Medical Papers/` for medical and bio-integration concept papers.

7. `Programs-and-Initiatives/Eden Initiative/` for initiative-level planning artifacts.

## Content Notes

1. The archive includes mixed formats (`.txt`, `.md`, `.docx`, `.pdf`, images, scripts).
2. Naming is intentionally heterogeneous and reflects evolving project drafts.
3. Some conceptual themes appear in more than one folder as parallel or copied iterations.
4. There is no single build system or executable application at the repository root.

## Maintenance Guidance

For future README updates after content changes:

1. Recount files with `find . -type f | wc -l`.
2. Rebuild extension counts with `find . -type f | sed 's|.*\.||' | tr '[:upper:]' '[:lower:]' | sort | uniq -c | sort -nr`.
3. Recheck top-level folders with `find . -mindepth 1 -maxdepth 1 -type d -printf '%f\n' | sort`.
